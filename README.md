# NASA-Project
a web that shows you the Nasa photo taken on an exact date
pick a date ,hit connect and it pulls the picture published that day using their Astronomy Picture of the Day (APOD) API
# Features
WebOS interface 
pulls the real NASA APOD image for any date you choose
shows NASA's official explanation text for each image
Photo credits (displays photographer copyright info when available)
Download button  save  any image straight from the app
move the scanner window anywhere on screen, just like a real desktop app
# How It Works
pick a date using the date input field then the app calls NASA's public APOD API
NASA returns that day's featured image, title, explanation, and if available photographer credit
The app displays everything inside a styled window and lets you download the image directly
tech stack:
Nasa APOD API,HTML, CSS, and JavaScript

No installation needed  just use it in a browser
download this repository
Open gy.html in any modern browser
Pick a date and click CONNECT TO APOD SERVER
